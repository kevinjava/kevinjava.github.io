Python中使用mysql数据库

ubuntu下安装mysql,python-mysqldb

sudo apt-get install mysql-client mysql-server
sudo apt-get install libmysqlclient-dev
sudo apt-get install python-mysqldb

根据提示输入密码即可。

使用python进行相关操作

导入包

import MySQLdb #注意大小写

创建数据库


    # encoding: utf-8
    #!/usr/bin/python

    import MySQLdb
    __author__ = 'kevliu'

    # 建立连接
    conn = MySQLdb.connect(host="localhost", user= "root", passwd= "123456")

    # 获取操作的游标
    cursor = conn.cursor()

    # 执行cursor语句创建数据库photo
    cursor.execute("""create database photo """)

    # 操作完毕，关闭游标
    cursor.close()

创建表


    # 选择photo db
    conn.select_db('photo')

    # 获取操作游标
    cursor = conn.cursor()

    # 执行插入表语句
    cursor.execute(""" create TABLE recodes(id varchar(100) UNIQUE , data VARCHAR(10000)) """)

    # 关闭游标
    cursor.close()

插入一条数据


    # dummy data
    value = ["test id", "test data"]

    cursor.execute("""insert INTO recodes VALUES (%s, %s)""", value)

    # 关闭游标
    cursor.close()

    # 提交数据
    conn.commit()

插入多条数据


    values = []
    for i in range(5):
        values.append(("test id" + str(i+1), "test data"+str(i+1)))
    # 使用executemany    
    cursor.executemany("""insert INTO recodes VALUES (%s, %s)""", values)

查询数据


    # 执行查询语句                                                       
    count = cursor.execute(""" select * from recodes """)          
    print str(count)                                               

    #获取游标中的第一条数据                                                   
    data = cursor.fetchone()                                       
    print 'ID: %s   info: %s' % data                               

    #获取游标中的5条数据                                                    
    datas = cursor.fetchmany(5)                                    
    for r in datas:                                                
        print r                                                    
    #重置游标位置，0,为偏移量，mode＝absolute | relative,默认为relative,           
    cursor.scroll(0,mode='absolute')                               
    #获取所有结果                                                        
    results = cursor.fetchall()                                    
    for r in results:                                              
        print r

解决mysql乱码问题

修改my.cnf配置文件 local: /etc/mysql/my.cnf

  1. 在[client]字段里加入default-character-set=utf8，如下：
    [client]
    port = 3306
    socket = /var/lib/mysql/mysql.sock
    default-character-set=utf8

  2. 在[mysqld]字段里加入character-set-server=utf8，如下：
    [mysqld]
    port = 3306
    socket = /var/lib/mysql/mysql.sock
    character-set-server=utf8

  3. 在[mysql]字段里加入default-character-set=utf8，如下：
    [mysql]
    no-auto-rehash
    default-character-set=utf8

  4. python连接是指定编码

    MySQLdb.connect(host=”localhost”, user= “root”, passwd= “123456”, charset=’utf8’, init_command=’SET NAMES UTF8’)
